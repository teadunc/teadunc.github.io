CSS Review Cards

What does HTML stand for?
HyperText Markup Language
What does CSS control?
The style and appearance of a webpage
What is Flexbox?
A CSS layout system for arranging items.
What is a class selector?
A selector used to style elements with a matching class.
What does hover do?
Applies styles when the mouse is over an element.
What is a pseudo-class?
A keyword added to selectors for special states.
What is a transition?
A smooth change between CSS property values.
What is an animation?
A sequence of changing styles over time.
